# 🧪 Atividade de Laboratório

Como prática deste módulo, cada aluno deverá realizar as seguintes atividades:

### 1. Explorar o DevTools (Ferramentas do Navegador)
- Acesse o DevTools (`F12` ou botão direito → "Inspecionar")
- Navegue pelas principais abas:
  - **Elements:** estrutura HTML e CSS
  - **Network:** requisições e respostas HTTP
  - **Console:** execução de scripts e erros
- Analise uma página real e identifique:
  - Estrutura do HTML
  - Arquivos carregados (CSS, JS, imagens)
  - Status das requisições (ex: 200, 404)

### 2. Criar e publicar um site pessoal no GitHub Pages
- Desenvolva uma página simples, conforme o exemplo da aula.
- O site deve conter:
  - Seu nome
  - Uma breve apresentação
  - Links ou informações relevantes
- Publique o site utilizando o **GitHub Pages**.

### 3. Compartilhar no Discord
- Envie o link do seu site no canal da disciplina.
- Explore os projetos dos colegas.
- *(Opcional)* Deixe feedbacks construtivos.
