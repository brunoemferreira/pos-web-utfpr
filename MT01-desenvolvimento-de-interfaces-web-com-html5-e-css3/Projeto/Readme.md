### 🎯 Objetivo
O objetivo deste projeto é desenvolver uma **Landing Page** de tema livre. Pode ser para um negócio da família, um portfólio pessoal ou um produto fictício. Você começará com o conteúdo dominado em aula e terminará explorando novos territórios, utilizando pesquisa e documentação para entregar um projeto com nível profissional de mercado.

---

### 📤 Entrega
A entrega deve ser realizada por meio de um **Link de repositório Git**.
1. Construa a sua aplicação.
2. Hospede-a em um **repositório público do GitHub**.
3. Envie o link do repositório nesta atividade.

---

### 📋 Checklist de Implementação
Os itens abaixo devem ser implementados de forma incremental. Na raiz do repositório, seu arquivo `Readme.md` deve conter o título do projeto, uma breve descrição e uma explicação de como cada ponto foi resolvido:

1.  **Estrutura Semântica Rigorosa** (Módulo 01)
    - Layout construído exclusivamente com tags semânticas para SEO e acessibilidade.
2.  **Sistema de Design com Variáveis** (Módulo 02)
    - Esquema de cores e tipografia centralizado em variáveis CSS globalmente.
3.  **Layout Híbrido (Flex & Grid)** (Módulo 03)
    - Combinação de Flexbox (menus/botões) e CSS Grid (grade principal).
4.  **Interface Responsiva** (Módulo 04)
    - Adaptação fluida de dispositivos mobile até resoluções desktop.
5.  **Microinterações de Feedback** (Módulo 05)
    - Animações de entrada e estados de hover para melhor experiência do usuário.
6.  **Curadoria de Código com IA** (Módulo 06)
    - Uso de IA para gerar/refatorar um componente específico, com documentação do ajuste.
7.  **Implementação de Dark Mode Nativo** (Pesquisa)
    - Uso da media query `prefers-color-scheme`.
8.  **Sticky Headers e Scroll Snap** (Pesquisa)
    - Cabeçalho fixo e seções com ajuste suave de scroll.
9.  **Otimização de Performance e Assets** (Pesquisa)
    - Uso de formatos modernos (WebP) e otimização de fontes (foco em Google PageSpeed).
10. **Acessibilidade Avançada com Teclado** (Pesquisa)
    - Gerenciamento de foco (`:focus-visible`) e atributos ARIA básicos.
